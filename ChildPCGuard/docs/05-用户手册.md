# 用户手册

**产品名称**: ChildPCGuard - 儿童电脑使用时间控制程序
**版本**: 1.0
**编写日期**: 2026-05-02

---

## 1. 产品介绍

### 1.1 产品概述

ChildPCGuard是一款运行在Windows平台上的儿童电脑使用时间控制程序，帮助家长管理孩子使用电脑的时间，防止沉迷电脑游戏。

### 1.2 主要功能

- 限制每日电脑使用总时长
- 连续使用一定时间后强制休息
- 定时自动关机
- 程序和网站使用监控
- 黑名单功能（禁止特定程序/网站）
- 进程保护（防止被结束）

### 1.3 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 专业版 |
| 内存 | 2GB以上 |
| 硬盘 | 100MB可用空间 |
| 权限 | 管理员权限（安装时需要） |

## 2. 安装指南

### 2.1 安装前准备

1. 确保拥有管理员账户
2. 关闭杀毒软件（可能拦截安装）
3. 确保网络连接正常（NTP功能需要）

### 2.2 安装步骤

1. 解压安装包到任意目录
2. 右键点击 `install.ps1`，选择"使用PowerShell运行"
3. 按照提示输入管理员密码
4. 等待安装完成

### 2.3 验证安装

安装完成后，可以通过以下方式验证：

1. 打开"服务"管理器（Win+R，输入services.msc）
2. 找到"Windows Security Update Service"
3. 确认服务状态为"正在运行"

## 3. 使用指南

### 3.1 配置管理

使用config.exe命令行工具管理配置：

```cmd
# 查看当前配置
config.exe --view

# 修改每日时长限制（分钟）
config.exe --set dailyLimit 120

# 修改连续使用限制（分钟）
config.exe --set continuousLimit 30

# 修改休息时长（分钟）
config.exe --set restDuration 10

# 修改关机时间（24小时制）
config.exe --set shutdownTime "21:00"

# 添加程序到黑名单
config.exe --block-app game.exe

# 从黑名单移除程序
config.exe --unblock-app game.exe

# 添加网站到黑名单
config.exe --block-site youtube.com

# 从黑名单移除网站
config.exe --unblock-site youtube.com

# 查看今日使用日志
config.exe --view-logs today

# 重置当日使用时间
config.exe --reset-daily
```

### 3.2 家长解锁

当孩子使用电脑时间用尽或触发黑名单时，电脑会锁定。

**解锁步骤**：
1. 屏幕上会显示锁屏界面
2. 输入家长密码
3. 按Enter键或点击"解锁"按钮

**紧急解锁**（如忘记密码）：
1. 连续按 `Ctrl + Alt + Shift + F12` 5次
2. 会弹出紧急对话框
3. 按提示操作

### 3.3 强制休息

当连续使用电脑达到限制时间后，会进入强制休息模式：
- 屏幕显示休息倒计时
- 倒计时期间无法使用电脑
- 倒计时结束后自动解锁

### 3.4 查看使用报告

```cmd
# 查看今日使用情况
config.exe --view-logs today

# 查看指定日期使用情况
config.exe --view-logs 2026-05-02
```

日志文件位于：`C:\ProgramData\ChildPCGuard\logs\`

## 4. 配置说明

### 4.1 默认配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 每日总时长限制 | 120分钟（工作日）/ 240分钟（周末） | 每日累计使用时间上限 |
| 连续使用限制 | 45分钟 | 连续使用多久后强制休息 |
| 休息时长 | 5分钟 | 强制休息的时间长度 |
| 关机时间 | 22:00 | 每天自动关机时间 |
| NTP时间验证 | 启用 | 是否验证系统时间 |

### 4.2 工作日与周末规则

程序自动区分工作日和周末：
- 工作日（周一至周五）：默认120分钟
- 周末（周六、周日）：默认240分钟

### 4.3 时间窗口（可选）

可以设置允许使用电脑的时间段：
- 例如：15:00-20:00
- 时间窗口外无法使用电脑

## 5. 常见问题

### 5.1 安装问题

**Q: 安装时提示"拒绝访问"**
A: 请确保以管理员身份运行PowerShell

**Q: 杀毒软件拦截安装**
A: 临时关闭杀毒软件后再安装，安装完成后可重新开启

### 5.2 使用问题

**Q: 忘记了家长密码**
A: 使用紧急解锁快捷键（Ctrl+Alt+Shift+F12 连续5次）

**Q: 孩子能通过任务管理器结束进程吗？**
A: 程序有进程保护机制，被结束后会自动重启

**Q: 重启电脑后程序还生效吗？**
A: 程序配置为开机自启动，重启后自动运行

**Q: 能通过安全模式绕过控制吗？**
A: 程序检测到安全模式启动会自动关机

### 5.3 卸载问题

**Q: 如何卸载程序？**
A: 以管理员身份运行 `uninstall.ps1`

**Q: 卸载后有残留吗？**
A: 正常卸载后会清除所有文件和注册的服务
