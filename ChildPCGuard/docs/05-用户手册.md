## 1. 产品介绍

### 1.1 产品概述

ChildPCGuard 是一款 Windows 平台的儿童电脑使用时间控制程序，帮助家长管理孩子使用电脑的时间，防止沉迷电脑游戏影响健康和学习。

### 1.2 适用场景

- 控制孩子每日使用电脑的总时长
- 防止长时间连续使用电脑
- 限制特定程序和网站的使用
- 设定每日自动关机时间

### 1.3 主要功能

| 功能 | 描述 |
|------|------|
| 时长限制 | 限制每日电脑使用总时长，工作日和周末可分别设置 |
| 强制休息 | 连续使用一定时间后，强制休息指定时长 |
| 定时关机 | 到达设定时间后自动关闭电脑 |
| 程序黑名单 | 禁止运行指定的程序 |
| 网站黑名单 | 禁止访问指定的网站 |
| 使用监控 | 记录程序和网站使用情况 |
| 进程保护 | 三层守护机制，防止监控进程被结束 |

### 1.4 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 专业版 |
| 运行环境 | .NET 8.0 |
| 内存 | 2GB 以上 |
| 硬盘 | 100MB 可用空间 |
| 权限 | 管理员权限（安装时需要） |
| 网络 | 可选（NTP 时间验证需要） |

## 2. 安装指南

### 2.1 安装前准备

1. 确认电脑运行 Windows 10/11 专业版
2. 准备一个管理员账户和密码
3. 确保已安装 .NET 8.0 运行时

### 2.2 安装步骤

**方法一：使用安装脚本**

1. 打开 PowerShell（管理员权限）
2. 切换到安装文件所在目录：
   ```powershell
   cd C:\path\to\install\files
   ```
3. 运行安装脚本：
   ```powershell
   .\install.ps1 -Password "your_password"
   ```
4. 等待安装完成

**方法二：手动安装**

1. 将可执行文件复制到 `C:\Program Files\ChildPCGuard\`
2. 打开 PowerShell（管理员权限）
3. 注册 Windows 服务：
   ```powershell
   New-Service -Name "WinSecSvc_a1b2c3d4" `
       -BinaryPathName "C:\Program Files\ChildPCGuard\ChildPCGuard.GuardService.exe" `
       -DisplayName "Windows Security Update Service" `
       -StartType Automatic
   ```
4. 启动服务：
   ```powershell
   Start-Service -Name "WinSecSvc_a1b2c3d4"
   ```

### 2.3 验证安装

安装完成后，验证以下步骤：

1. 打开"服务"管理器（Win+R，输入 `services.msc`）
2. 找到"Windows Security Update Service"
3. 确认服务状态为"正在运行"
4. 打开任务管理器，确认存在以下进程：
   - svchost.exe（AgentA）
   - RuntimeBroker.exe（AgentB）

### 2.4 安装后配置

安装完成后，配置文件位于：
```
C:\ProgramData\ChildPCGuard\config.json
```

默认配置：
- 工作日每日限制：120 分钟
- 周末每日限制：240 分钟
- 连续使用限制：45 分钟
- 强制休息时长：5 分钟
- 关机时间：22:00

## 3. 使用指南

### 3.1 家长操作

#### 解锁电脑

当孩子使用电脑时间用尽或触发黑名单时，屏幕会显示锁屏界面。

**解锁步骤**：
1. 在锁屏界面输入家长密码
2. 按 Enter 键或点击"解锁"按钮

**紧急解锁**（忘记密码时）：
1. 连续按 `Ctrl + Alt + Shift + F12` 5 次
2. 弹出紧急对话框，显示安装时设定的密码

#### 修改配置

配置文件位于 `C:\ProgramData\ChildPCGuard\config.json`。

使用文本编辑器打开并修改：

```json
{
    "Rules": {
        "Weekdays": {
            "DailyLimitMinutes": 120,
            "AllowedTimeWindows": [
                { "Start": "15:00", "End": "20:00" }
            ]
        },
        "Weekends": {
            "DailyLimitMinutes": 240,
            "AllowedTimeWindows": [
                { "Start": "09:00", "End": "21:00" }
            ]
        }
    },
    "ContinuousLimitMinutes": 45,
    "RestDurationMinutes": 5,
    "AutoShutdownTime": "22:00",
    "BlockedApps": ["game.exe"],
    "BlockedSites": ["youtube.com"]
}
```

修改后保存，服务会在下一个检测周期自动应用新配置。

#### 添加程序黑名单

```json
{
    "BlockedApps": ["game.exe", "minecraft.exe"]
}
```

#### 添加网站黑名单

```json
{
    "BlockedSites": ["youtube.com", "bilibili.com"]
}
```

#### 查看使用记录

日志文件位于 `C:\ProgramData\ChildPCGuard\logs\`。

- 程序日志：`YYYY-MM-DD_process.json`
- 网站日志：`YYYY-MM-DD_web.json`

示例程序日志：
```json
{
    "Date": "2026-05-02",
    "Records": [
        {
            "Timestamp": "2026-05-02T15:30:00",
            "ProcessName": "chrome",
            "ProcessPath": "C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe",
            "Duration": 3600
        }
    ]
}
```

### 3.2 儿童用户视角

#### 正常使用

孩子正常开机后，系统自动开始监控使用时间。

#### 强制休息

当连续使用电脑达到限制时间后：
1. 屏幕显示锁屏界面
2. 显示剩余休息时间的倒计时
3. 倒计时期间无法使用电脑
4. 倒计时结束后自动解锁

#### 时间用尽

当每日使用总时长达到限制后：
1. 屏幕显示锁屏界面
2. 显示"今天的使用时间已到"提示
3. 输入家长密码后可继续使用

#### 黑名单触发

当运行被禁止的程序或访问被禁止的网站时：
1. 屏幕显示锁屏界面
2. 显示"程序已阻止"或"网站已阻止"提示
3. 输入家长密码后可继续使用

## 4. 配置说明

### 4.1 配置项列表

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `IsEnabled` | bool | true | 是否启用控制 |
| `AdminPasswordHash` | string | - | 家长密码哈希值 |
| `Rules.Weekdays.DailyLimitMinutes` | int | 120 | 工作日每日限制（分钟） |
| `Rules.Weekends.DailyLimitMinutes` | int | 240 | 周末每日限制（分钟） |
| `ContinuousLimitMinutes` | int | 45 | 连续使用限制（分钟） |
| `RestDurationMinutes` | int | 5 | 强制休息时长（分钟） |
| `AutoShutdownTime` | string | "22:00" | 每日关机时间 |
| `WarningMinutes` | int[] | [10, 5, 1] | 关机前警告分钟数 |
| `IdleThresholdMs` | int | 5000 | 空闲阈值（毫秒） |
| `BlockedApps` | string[] | [] | 程序黑名单 |
| `BlockedSites` | string[] | [] | 网站黑名单 |
| `UseNtpValidation` | bool | true | 是否启用 NTP 验证 |
| `NtpServers` | string[] | ["pool.ntp.org"] | NTP 服务器列表 |
| `NtpToleranceMinutes` | int | 5 | NTP 容差（分钟） |

### 4.2 时间窗口配置

时间窗口限制允许使用的时段：

```json
{
    "Rules": {
        "Weekdays": {
            "DailyLimitMinutes": 120,
            "AllowedTimeWindows": [
                { "Start": "15:00", "End": "20:00" }
            ]
        }
    }
}
```

- 时间窗口外使用电脑会立即锁屏
- 可设置多个时间段
- 窗口格式为 "HH:mm"（24 小时制）

### 4.3 密码设置

安装时设定的密码会使用 SHA-256 哈希算法存储到配置文件中。

如需修改密码，需要重新运行安装脚本：

```powershell
.\install.ps1 -Password "new_password"
```

## 5. 常见问题

### 5.1 安装问题

**Q: 安装时提示"拒绝访问"**

A: 请确保以管理员身份运行 PowerShell。右键点击 PowerShell，选择"以管理员身份运行"。

**Q: 杀毒软件拦截安装**

A: 临时关闭杀毒软件后再安装，安装完成后可重新开启。ChildPCGuard 是合法的监控软件。

**Q: .NET 8.0 未安装**

A: 从微软官网下载并安装 .NET 8.0 Runtime：https://dotnet.microsoft.com/download/dotnet/8.0

### 5.2 使用问题

**Q: 忘记了家长密码**

A: 使用紧急解锁：连续按 `Ctrl + Alt + Shift + F12` 5 次，会弹出紧急对话框显示密码。

**Q: 孩子能通过任务管理器结束进程吗？**

A: 不能。程序有三层守护机制，被结束后会自动重启。

**Q: 重启电脑后程序还生效吗？**

A: 是的。程序配置为开机自启动，重启后自动运行。

**Q: 能通过安全模式绕过控制吗？**

A: 不能。程序检测到安全模式启动会自动关机。

**Q: 如何临时禁用控制？**

A: 修改配置文件 `IsEnabled` 为 `false`，或运行 `config.exe --pause`。

### 5.3 卸载问题

**Q: 如何卸载程序？**

A: 以管理员身份运行卸载脚本：
```powershell
.\uninstall.ps1 -Password "your_password"
```

**Q: 卸载后如何保留使用记录？**

A: 卸载前备份 `C:\ProgramData\ChildPCGuard\logs\` 目录。

**Q: 卸载后有残留吗？**

A: 正常卸载会清除所有文件和注册的服务。如需保留配置文件，卸载前备份 `config.json`。

## 6. 技术支持

如遇到问题，请检查以下内容：

1. 服务是否正常运行：`Get-Service -Name "WinSecSvc_a1b2c3d4"`
2. Agent 进程是否存在：任务管理器中查看 svchost.exe 和 RuntimeBroker.exe
3. 日志文件是否有错误记录：`C:\ProgramData\ChildPCGuard\logs\`
