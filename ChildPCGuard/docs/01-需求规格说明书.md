# 需求规格说明书

**项目名称**: ChildPCGuard - 儿童电脑使用时间控制程序
**版本**: 1.0
**编写日期**: 2026-05-02

---

## 1. 项目背景和目标

### 1.1 项目背景

家长希望限制孩子（初一学生，具备一定技术能力，能够通过任务管理器结束进程）使用电脑的时间，防止沉迷电脑游戏影响健康和学习。

### 1.2 项目目标

开发一个Windows平台下的儿童电脑使用时间控制程序，实现以下目标：
- 限制每日电脑使用总时长
- 每连续使用45分钟，强制休息5分钟
- 每天22:00自动关机
- 程序完全隐藏，不易被发现和关闭
- 记录程序和网站使用情况
- 支持程序和网站黑名单功能

### 1.3 使用场景

| 角色 | 使用场景 |
|------|----------|
| 孩子用户 | 使用电脑，被时间控制，无法结束监控进程 |
| 家长用户 | 设置控制策略，解锁电脑，查看使用报告 |

## 3. 非功能需求

### 3.1 性能需求

| 指标 | 要求 |
|------|------|
| CPU占用 | 正常运行<1%，峰值<5% |
| 内存占用 | <50MB |
| 启动时间 | <5秒 |
| 锁屏响应 | 触发后<2秒显示锁屏 |

### 3.2 可靠性需求

| 指标 | 要求 |
|------|------|
| 运行稳定性 | 7x24小时稳定运行 |
| 进程存活率 | 守护进程被结束后可自动恢复 |
| 数据完整性 | 断电或异常重启后数据不丢失 |

### 3.3 安全性需求

| 指标 | 要求 |
|------|------|
| 密码强度 | 至少6位字母数字组合 |
| 加密算法 | AES-256-CBC |
| 数据存储 | 本地存储，不上传网络 |

### 3.4 兼容性需求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 专业版 |
| .NET版本 | .NET 8 |
| 浏览器支持 | Chrome、Edge、Firefox历史记录读取 |

## 4. 用例图和用户故事

### 4.1 参与者

| 参与者 | 描述 |
|--------|------|
| 孩子用户 | 电脑的受限使用者，受时间控制 |
| 家长用户 | 管理者，配置控制策略，解锁电脑 |

### 4.2 用例列表

| 用例ID | 用例名称 | 参与者 | 描述 |
|--------|----------|--------|------|
| UC01 | 使用电脑 | 孩子用户 | 孩子正常使用电脑，时间被监控和记录 |
| UC02 | 时间限制触发 | 孩子用户 | 到达时间限制后被锁屏 |
| UC03 | 强制休息 | 孩子用户 | 连续使用45分钟后进入强制休息倒计时 |
| UC04 | 密码解锁 | 家长用户 | 输入密码解锁电脑 |
| UC05 | 查看使用报告 | 家长用户 | 查看程序和网站使用日志 |

| UC08 | 定时关机 | 系统 | 到达指定时间自动关机 |

### 4.3 用户故事

**作为家长用户，我想要：**
- 设置孩子每天使用电脑的总时长
- 设置每天的关机时间
- 添加和删除禁止的程序
- 添加和删除禁止的网站
- 查看孩子的电脑使用情况
- 在忘记密码时能够解锁

**作为系统，我需要：**
- 监控孩子使用电脑的时间
- 防止孩子通过任务管理器结束监控进程
- 在到达限制时自动锁屏
- 在指定时间自动关机

## 6. 验收标准

### 6.1 功能验收

| 需求ID | 验收条件 | 测试方法 |
|--------|----------|----------|
| F001 | 每日累计使用达到限制后无法继续使用电脑 | 配置10分钟限制，使用10分钟后验证锁屏 |
| F002 | 连续使用45分钟后强制休息5分钟 | 配置45分钟连续限制，验证强制休息触发 |
| F003 | 到达22:00自动关机 | 配置22:00关机，验证执行 |
| F010 | 守护进程被结束后可自动重启 | 手动结束Agent进程，验证自动恢复 |
| F012 | 进程伪装成功 | 检查任务管理器中进程名称 |
| F021 | 正确密码可解锁 | 输入正确密码验证解锁 |
| F023 | 密码错误3次后锁定5分钟 | 连续输入错误密码验证 |
| F024 | 紧急解锁可用 | 按Ctrl+Alt+Shift+F12 5次验证 |

- 正常运行时CPU占用<1%

- 锁屏响应时间<2秒

### 6.3 可靠性验收

- 连续运行24小时无崩溃
