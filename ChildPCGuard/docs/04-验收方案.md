## 1. 验收标准和验收准则

### 1.2 验收准则

**功能验收准则**：
- 所有功能需求（F001-F053）必须实现
- 所有验收测试用例必须通过
- 功能必须与需求规格说明书一致

**性能验收准则**：
- 正常运行时CPU占用 < 1%
- 内存占用 < 50MB
- 锁屏响应时间 < 2秒

**可靠性验收准则**：
- 守护进程被结束后5秒内自动恢复
- 连续运行24小时无崩溃
- 断电恢复后数据不丢失

**安全性验收准则**：
- 密码加密存储
- 进程无法通过常规方式结束
- 时间篡改可被检测

## 3. 验收流程

### 3.1 验收准备

1. 确认测试环境就绪（Windows 10/11专业版虚拟机）
2. 确认所有源代码和文档已提交
3. 确认编译通过并生成可执行文件
4. 准备验收检查表

### 3.2 验收执行

**阶段一：安装验收（30分钟）**
1. 执行全新安装
2. 检查服务注册
3. 检查进程伪装
4. 验证数据目录创建

**阶段二：功能验收（2小时）**
1. 时间控制功能测试
2. 进程保护功能测试
3. 锁屏控制功能测试
4. 黑名单功能测试
5. 配置管理功能测试

**阶段三：性能验收（30分钟）**
1. CPU占用测试
2. 内存占用测试
3. 锁屏响应时间测试

**阶段四：可靠性验收（24小时）**
1. 长时间运行测试
2. 进程恢复测试

**阶段五：卸载验收（30分钟）**
1. 执行卸载
2. 检查残留文件
3. 验证卸载完整性

### 3.3 验收记录

每项测试完成后记录：
- 测试时间
- 测试人员
- 测试结果（通过/失败）
- 问题描述（如有）
