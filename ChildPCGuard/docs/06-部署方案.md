## 1. 部署架构

```

│  │              WinSecSvc_a1b2c3d4                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                              │                              │
│          ┌───────────────────┼───────────────────┐        │
│          ▼                   ▼                   ▼        │
│    ┌──────────┐       ┌──────────┐       ┌──────────┐      │
│    │AgentA    │◄─────►│AgentB    │       │LockOverlay│      │
```

## 2. 部署流程

### 2.1 部署前检查

| 检查项 | 要求 | 检查方法 |
|--------|------|----------|
| 操作系统版本 | Windows 10/11专业版 | 设置 → 系统 → 关于 |
| 管理员权限 | 当前用户为管理员 | 命令行输入 whoami |
| .NET 8运行时 | 已安装 | dotnet --list-runtimes |
| 磁盘空间 | 至少100MB可用 | 磁盘属性查看 |

### 2.2 安装步骤

#### 步骤1：准备工作

```powershell
# 以管理员身份打开PowerShell
# 进入安装目录
cd C:\Path\To\ChildPCGuard
```

#### 步骤2：执行安装

```powershell
# 执行安装脚本
.\install.ps1
```

安装脚本自动完成：
1. 创建数据目录
2. 生成加密密钥
3. 创建初始配置文件
4. 复制伪装进程到System32
5. 注册Windows服务
6. 配置服务自动重启
7. 启动服务

#### 步骤3：验证安装

```powershell
# 检查服务状态
Get-Service -Name WinSecSvc_a1b2c3d4

# 检查进程
Get-Process | Where-Object {$_.ProcessName -eq "svchost" -or $_.ProcessName -eq "RuntimeBroker"}
```

#### 步骤4：配置初始密码

```cmd
# 首次运行需要设置家长密码
# （通过配置文件直接修改）
notepad C:\ProgramData\ChildPCGuard\config.json
```

### 2.3 部署参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ServiceName | WinSecSvc_a1b2c3d4 | Windows服务名 |
| DisplayName | Windows Security Update Service | 服务显示名 |
| DataDirectory | C:\ProgramData\ChildPCGuard | 数据目录 |
| AgentAPath | C:\Windows\System32\svchost.exe | AgentA伪装路径 |
| AgentBPath | C:\Windows\System32\RuntimeBroker.exe | AgentB伪装路径 |

## 3. 回滚方案

### 3.1 回滚触发条件

| 场景 | 触发回滚 |
|------|----------|
| 安装后服务无法启动 | 是 |
| 配置文件损坏 | 是 |
| 进程伪装失败 | 是 |
| 功能异常影响使用 | 是 |

### 3.2 回滚步骤

#### 自动回滚

```powershell
# 执行卸载脚本
.\uninstall.ps1
```

卸载脚本自动完成：
1. 停止服务
2. 删除服务注册
3. 删除伪装进程
4. 清理计划任务
5. （可选）删除数据目录

#### 手动回滚

如果自动回滚失败，执行以下步骤：

```powershell
# 1. 停止服务
net stop WinSecSvc_a1b2c3d4

# 2. 删除服务
sc delete WinSecSvc_a1b2c3d4

# 3. 删除伪装进程
Remove-Item C:\Windows\System32\svchost.exe -Force
Remove-Item C:\Windows\System32\RuntimeBroker.exe -Force

# 4. 删除数据目录
Remove-Item C:\ProgramData\ChildPCGuard -Recurse -Force
```

### 3.3 回滚后验证

| 验证项 | 预期结果 |
|--------|----------|
| 服务列表 | WinSecSvc_a1b2c3d4不存在 |
| 进程列表 | svchost.exe和RuntimeBroker.exe正常（系统原生） |
| 数据目录 | C:\ProgramData\ChildPCGuard不存在 |

## 4. 应急响应

### 4.1 服务无法启动

**症状**：服务状态为"已停止"，启动失败

**排查步骤**：
1. 检查事件日志：`Get-EventLog -LogName Application -Source WinSecSvc`
2. 检查配置文件：`C:\ProgramData\ChildPCGuard\config.json`是否存在且有效
3. 检查依赖：`sc queryex type= service state= all | findstr WinSecSvc`

**解决方案**：
```powershell
# 重新注册服务
sc delete WinSecSvc_a1b2c3d4
.\install.ps1
```

### 4.2 进程不断重启

**症状**：Agent进程反复退出重启

**排查步骤**：
1. 检查两个Agent进程是否同时存在
2. 检查心跳超时设置
3. 检查服务与Agent通信

**解决方案**：
```powershell
# 重启服务
net stop WinSecSvc_a1b2c3d4
net start WinSecSvc_a1b2c3d4
```

### 4.3 锁屏界面无法显示

**症状**：触发锁屏条件但无锁屏界面

**排查步骤**：
1. 检查LockOverlay.exe是否存在
2. 检查LockOverlay.exe执行权限
3. 检查桌面切换API是否正常

**解决方案**：
```powershell
# 手动触发锁屏测试
C:\Windows\System32\LockOverlay.exe 1
```

### 4.4 忘记密码

**解决方案**：
1. 使用紧急解锁：`Ctrl + Alt + Shift + F12` 连续5次
2. 安全模式卸载重装
