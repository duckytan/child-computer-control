## 二、技术方案要点

### 2.1 推荐架构

```
Windows服务 + 双守护进程 + 锁屏界面
```

### 2.2 核心功能

1. **时间监控**: 使用GetLastInputInfo统计实际使用时间
2. **进程保护**: 双进程互相守护 + 服务监控
3. **锁屏**: 自定义桌面 + 密码验证
4. **定时关机**: 每天22:00执行shutdown命令

### 2.3 技术选型

- **语言**: C# (.NET 8)
- **架构**: Windows Service + 用户态进程
- **通信**: Named Pipe
- **配置**: JSON文件 + 注册表备份
