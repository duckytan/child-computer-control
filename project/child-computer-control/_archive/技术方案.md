# 儿童电脑使用时间控制程序 - 技术方案

## 一、项目概述

### 1.1 项目背景

家长希望限制孩子使用电脑的时间，防止沉迷电脑游戏影响健康和学习。孩子已有5年的电脑使用经验，具备一定的技术能力，能够通过任务管理器结束进程。

### 1.2 功能目标

| 功能 | 描述 | 达成标准 |
|------|------|----------|
| 使用时间限制 | 每日累计使用时长上限 | 达到上限自动锁屏 |
| 定时关机 | 每天22:00强制关机 | 不可绕过 |
| 进程保护 | 防止监控进程被结束 | 双进程互相守护 |
| 隐秘运行 | 程序不易被发现 | 无窗口、无托盘图标 |
| 密码保护 | 家长可解锁和配置 | 密码验证 |

## 三、详细设计

### 3.1 ChildControlService (Windows服务)

**功能**:
- 开机自启动
- 持续监控电脑使用时间
- 管理守护进程AgentA和AgentB
- 执行定时关机任务
- 与各进程通信协调

**核心代码逻辑**:

```csharp
public class ChildControlService : ServiceBase
{
    private Timer _usageCheckTimer;
    private TimeTracker _timeTracker;
    private ProcessGuardian _processGuardian;
    private ShutdownScheduler _shutdownScheduler;

    protected override void OnStart(string[] args)
    {
        // 初始化组件
        _timeTracker = new TimeTracker();
        _processGuardian = new ProcessGuardian();
        _shutdownScheduler = new ShutdownScheduler();

        // 启动守护进程
        _processGuardian.StartGuardians();

        // 启动使用时间检查 (每30秒)
        _usageCheckTimer = new Timer(CheckUsage, null, 0, 30000);

        // 启动关机调度
        _shutdownScheduler.Start();
    }

    private void CheckUsage(object state)
    {
        if (_timeTracker.IsLimitReached())
        {
            // 触发锁屏
            TriggerLockScreen();
        }
    }
}
```

**设计原理**:

**Agent.exe 实现**:

```csharp
public class Agent
{
    private readonly string _partnerName;
    private Process _partnerProcess;
    private Timer _heartbeatTimer;
    private readonly string _pipeName;

    public Agent(string partnerName, string pipeName)
    {
        _partnerName = partnerName;
        _pipeName = pipeName;
    }

    public void Start()
    {
        // 启动配对进程
        StartPartnerProcess();

        // 互相监控定时器
        _heartbeatTimer = new Timer(MonitorPartner, null, 0, 3000);

        // 通过命名管道向Service发送心跳
        SendHeartbeat();

        // 等待被结束
        Thread.Sleep(Timeout.Infinite);
    }

    private void MonitorPartner(object state)
    {
        if (_partnerProcess == null || _partnerProcess.HasExited)
        {
            // 配对进程已退出，通知Service
            NotifyServicePartnerDead();
            // 重新启动配对进程
            StartPartnerProcess();
        }
    }

    private void SendHeartbeat()
    {
        using (var client = new NamedPipeClientStream(".", _pipeName))
        {
            client.Connect(1000);
            // 发送心跳包
        }
    }
}
```

### 3.3 时间追踪器 (TimeTracker)

**实现原理**:

使用 `GetLastInputInfo` Win32 API 获取用户最后一次输入的时间，通过计算两次检测的时间差来累计用户实际使用时间。

```csharp
public class TimeTracker
{
    private TimeSpan _dailyLimit;
    private TimeSpan _usedTime = TimeSpan.Zero;
    private DateTime _dayStart;
    private uint _lastInputTick;

    [DllImport("user32.dll")]
    static extern bool GetLastInputInfo(ref LASTINPUTINFO plii);

    [StructLayout(LayoutKind.Sequential)]
    struct LASTINPUTINFO
    {
        public uint cbSize;
        public uint dwTime;
    }

    public void Update()
    {
        LASTINPUTINFO lastInput = new LASTINPUTINFO();
        lastInput.cbSize = (uint)Marshal.SizeOf(lastInput);

        if (GetLastInputInfo(ref lastInput))
        {
            uint currentTick = (uint)Environment.TickCount;
            uint idleMs = currentTick - lastInput.dwTime;

            // 如果距离上次输入小于5秒，认为用户正在使用
            if (idleMs < 5000)
            {
                _usedTime += TimeSpan.FromMilliseconds(idleMs);
            }
        }
    }

    public bool IsLimitReached()
    {
        return _usedTime >= _dailyLimit;
    }
}
```

### 3.4 锁屏模块

**锁屏流程**:

```
1. Service检测到超时
2. 通知LockScreen进程启动
3. LockScreen创建隐藏桌面
4. SwitchDesktop到锁屏桌面
5. 显示密码输入界面
6. 验证密码正确则切换回原桌面
```

**LockScreen.exe 实现**:

```csharp
public partial class LockScreenWindow : Window
{
    [DllImport("user32.dll")]
    static extern bool SwitchDesktop(IntPtr hDesktop);

    private IntPtr _originalDesktop;
    private IntPtr _lockDesktop;

    public LockScreenWindow()
    {
        InitializeComponent();

        // 保存当前桌面
        _originalDesktop = GetThreadDesktop(GetCurrentThreadId());

        // 创建锁屏桌面
        _lockDesktop = CreateDesktop("LockScreen_" + Guid.NewGuid().ToString(),
            IntPtr.Zero, IntPtr.Zero, 0,
            DESKTOP_CREATEWINDOW | DESKTOP_SWITCHDESKTOP,
            IntPtr.Zero);

        // 切换到锁屏桌面
        SwitchDesktop(_lockDesktop);
    }

    private void UnlockButton_Click(object sender, RoutedEventArgs e)
    {
        if (VerifyPassword(PasswordBox.Password))
        {
            // 切换回原桌面
            SwitchDesktop(_originalDesktop);

            // 通知Service解锁
            NotifyServiceUnlocked();

            // 关闭锁屏
            Close();
        }
        else
        {
            // 密码错误提示
            ErrorMessage.Visibility = Visibility.Visible;
        }
    }
}
```

### 3.5 定时关机调度

**实现**:

```csharp
public class ShutdownScheduler
{
    private Timer _checkTimer;
    private const int ShutdownHour = 22;
    private const int ShutdownMinute = 0;
    private bool _shutdownInitiated = false;

    public void Start()
    {
        _checkTimer = new Timer(CheckShutdownTime, null, 0, 60000);
    }

    private void CheckShutdownTime(object state)
    {
        if (_shutdownInitiated) return;

        var now = DateTime.Now;
        if (now.Hour == ShutdownHour && now.Minute == ShutdownMinute)
        {
            _shutdownInitiated = true;

            // 先锁屏提醒
            TriggerLockScreenWithMessage("即将关机，请保存工作...");

            // 等待30秒后关机
            Task.Delay(30000).ContinueWith(_ =>
            {
                ExecuteShutdown();
            });
        }
    }

    private void ExecuteShutdown()
    {
        // 使用shutdown命令强制关机
        Process.Start(new ProcessStartInfo
        {
            FileName = "shutdown",
            Arguments = "/s /f /t 0",
            UseShellExecute = false,
            CreateNoWindow = true
        });
    }
}
```

## 六、部署方案

### 6.1 安装流程

1. 以管理员身份运行安装程序
2. 创建Windows服务 `ChildControlService`
3. 生成配置密码
4. 设置服务自动启动
5. 启动服务

### 6.2 安装命令

```batch
:: 安装服务
sc create ChildControlService binPath= "C:\Program Files\ChildControl\ChildControl.Service.exe" start= auto

:: 配置服务失败后自动重启
sc failure ChildControlService reset= 86400 actions= restart/60000/restart/60000/restart/60000

:: 启动服务
net start ChildControlService
```

## 七、应急处理

### 7.1 紧急解锁

连续按 `Ctrl + Alt + Shift + F12` 5次，系统会弹出紧急解锁对话框。

### 7.2 安全模式

在安全模式下程序不会运行，可以通过安全模式修改配置。

### 7.3 卸载

通过Windows设置 -> 应用 -> 卸载，或运行 uninstall.bat。

## 八、技术风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 被结束进程 | 失去监控 | 双进程互相守护 + 服务监控 |
| 安全模式绕过 | 完全失效 | 配合BIOS密码 |
| 卸载程序 | 失去控制 | 隐藏卸载入口 |
| 系统还原 | 配置丢失 | 配置存储在注册表 |
