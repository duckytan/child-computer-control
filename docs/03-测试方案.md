```mermaid
stateDiagram-v2
    [*] --> Normal
    Normal --> Using: RecordActivity
    Using --> Locked: TotalTime >= DailyLimit
    Using --> Resting: ContinuousTime >= 45min
    Resting --> Using: RestTime >= 5min
    Locked --> Using: PasswordCorrect
    Using --> Shutdown: Time >= AutoShutdown
```

| 测试场景 | 初始状态 | 事件 | 预期状态 |

| 强制休息 | Using | ContinuousTime >= 45min | Resting |
| 休息结束 | Resting | RestTime >= 5min | Using |
| 密码解锁 | Locked | 密码正确 | Using |
| 定时关机 | Using | Time >= AutoShutdown | Shutdown |
